# simplarr
1. ```bash ./simplarr.sh```
2. Enter API keys
3. Add root folders:
Sonarr:`/media/shows/`
Radarr:`/media/movies/`
4. Add torrent blackhole download client:
Torrent Folder: `/media/blackhole/*arr/`.
Watch Folder: `/media/blackhole/*arr/completed`.
5. Add Sonarr and Radarr apps to Prowlarr.
6. Add indexers to Prowlarr.
